我们还可以使用计算属性绑定类:

<div :class="classObject"></div>

  isActive: true,
  error: null
},
computed: {
  classObject(){
    return {
      active: this.isActive && !this.error
    }
  }
}

# 使用数组

当我们要添加的类不止一个时, 我们也可以用数组的方式绑定类:

<div :class="[class1, class2]></div>
1
data: {
  class1: 'active',
  class2: 'text-dander'
}

// <div class="active text-danger"></div>
